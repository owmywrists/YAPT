# Yet Another PathTracer

Features:
- KD Trees
- Materials: emissive, lambertian, mirror, mix
- Interactive rendering
- Interpolated normals (vertex normals)
- Multithreading with OpenMP
- UV texture mapping
- Fresnel reflection
- DoF
- OBJ parsing
- Immediate mode GUI (Dear ImGui)
-
